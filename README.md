## 配置

打开命令面板快捷键: `shift + ctrl + p`

### 用户设置

> 多行编辑和自定义快捷键

常用快捷键:

1. 复制光标所在行： `ctrl + c`
2. 替换文本: `ctrl + h`
3. 光标所在行的上方插入一行: `shift + ctrl + enter`
4. 光标所在行的下方插入一行: `ctrl + enter`
5. 文件夹内查找: `shift + ctrl + f`
6. 文档格式化: `shift + alt + f`
